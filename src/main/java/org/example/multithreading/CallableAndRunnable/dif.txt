Основные различия между Callable & Runnable:

* Runnable.run() не может возвращать результат или выбрасывать проверяемое исключение,
  в то время как Callable.call() может.

* Runnable используется для задач, которые выполняются асинхронно без ожидания результата,
  в то время как Callable используется для задач, которые должны вернуть результат.

* Для выполнения Callable обычно используется ExecutorService,
  который возвращает объект Future, предоставляющий методы для извлечения результата и других операций.


 Future

  Future — это интерфейс в Java, представляющий результат асинхронной вычислительной задачи.

  Он предоставляет методы для проверки, завершена ли задача, для получения результата задачи и для обработки исключений,
  возникших во время выполнения задачи.

  2 метода из Основных для примера интерфейса Future:

  get():
  Возвращает результат выполнения задачи. Этот метод блокируется до тех пор, пока задача не будет завершена.

  Если поток вызывает метод get() на объекте Future, и соответствующая задача еще не завершена,
  Этот поток будет "заблокирован" или "приостановлен" и не продолжит выполнение, пока задача не завершится и результат не станет доступным (или пока не будет выброшено исключение).

  Это поведение может быть полезным, когда вам нужно дождаться завершения асинхронной задачи перед тем, как продолжить выполнение основного потока.
